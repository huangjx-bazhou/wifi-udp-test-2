# wifi-udp-test-2
维拓启创ESP-12F wifi模块UDP协议测试

当前版本使用 mDNS 发布 UDP 服务，设备连网后会注册：

- 主机名：ir24-device.local
- 服务类型：_ir24._udp
- 端口：8888

服务 TXT 会附带：

- device=24-channel-ir

## 传输协议

- 网络层：Wi-Fi（STA 模式接入局域网）
- 传输层：UDP
- 业务端口：8888（设备监听）
- 服务发现：mDNS（主机名 ir24-device.local，服务 _ir24._udp）

## 通信协议

当前实现为“请求-响应”模式：

1. 上位机向设备 UDP 8888 端口发送请求帧（固定 8 字节）。
2. 设备收到后回包到请求来源 IP/端口，响应帧固定 224 字节。

### 请求帧（上位机 -> 设备）

- 长度：8 字节
- 用途：触发设备发送一次 224 字节采样数据帧

### 响应帧（设备 -> 上位机）

- 长度：224 字节
- 字节 1~4：固定帧头 `84 6F 0A 00`
- 字节 5~8：递增包计数（uint32，大端序）
- 字节 9~224：当前版本填 0（预留数据区）
